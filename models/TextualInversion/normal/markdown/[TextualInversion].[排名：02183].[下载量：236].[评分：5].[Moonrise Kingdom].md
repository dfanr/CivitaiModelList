### 三、详情
<p>This is my second model related to Wes Anderson, following the success of my previous Grand Budapest Hotel style model. With this Moonrise Kingdom model, I hope to provide artists and enthusiasts with a new tool to create beautiful and unique artwork in the distinctive style of Wes Anderson's films.</p><p></p><p>This AI drawing model is based on movie screenshots from Moonrise Kingdom and can be used to create images with a similar style. The model was created using the SD1.5 model and can be used with other checkpoints based on SD1.5. The input image size for this model is 600x320, but standard sizes can also be selected when generating the image.
