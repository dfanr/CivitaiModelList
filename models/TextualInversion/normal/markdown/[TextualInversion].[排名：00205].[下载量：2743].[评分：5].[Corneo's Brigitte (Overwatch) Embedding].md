### 三、详情
<p>This is a textual inversion embedding for Brigitte (Lindholm) from Overwatch.</p><p>Use <code>corneo_brigitte</code> to activate. The eyes are pretty bad in the embedding by itself, so you'll definitely want to prompt for the eyes to be a bit better, or just inpaint them after generating the rest of the image. I don't think this one works quite as well as my other character embeddings but it was a request so here it is!
