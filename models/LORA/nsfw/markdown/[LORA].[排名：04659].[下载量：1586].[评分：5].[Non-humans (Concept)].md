### 三、详情
<p><strong>Warning: the dataset for this included a lot of NSFW images so beware and use at your own discretion!!</strong></p><p>This is my attempt at a LoRA which helps producing images of all sorts of non-human species, such as animals, creatures, monsters, demons, Pokemon, and much more. I found that existing anime models aren't very good in handling those, and will either mix the creature traits with human traits, or generate something completely incoherent.</p><p>This was trained using "animefull-final-pruned" checkpoint and should work with most Anime style models. Using realistic models was not tested and is probably a bad idea. Dataset was mostly extracted from e621 so you should be able to prompt with the tags found there (e.g. mammal, pokemon, lagomorph, monster etc.)</p><p></p>
