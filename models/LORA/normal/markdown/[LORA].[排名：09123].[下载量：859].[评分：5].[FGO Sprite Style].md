### 三、详情
<p>This LoRA allows you to generate characters in the style of a Fate/Grand Order game sprite.</p><p></p><p>The LoRA was tested on various models, and the results were generally pretty unsatisfactory when used alone. I recommand using my other LoRA to replicate Ufotable artstyle, and, if you're using Automatic1111 webui, to turn on Highres. Fix ! With the Ufotable Lora and the Highres Fix, the generated images are much better than using the FGO Sprite LoRA alone. You can see the difference with the images of this post.</p><p></p><p>While using this LoRA, I recommand adding the <code>fgo sprite</code> tag to your prompt, to ensure to have only FGO Sprite style images.</p><p>The LoRA seems to give better results while using AnythingV5, but models like AnythingV4.5, AnythingV3 or NAI also work. It's up to you, to use the model you like the most.</p><p></p><p>The LoRA weight should be between 0.8 and 1.0, and I recommand using a 9:16 ratio, or something resembling it, since using a 1:1 ratio to generate an image let pretty much every model add some (really cool) effects in the background, or to simply generate multiple times the same character.</p>
