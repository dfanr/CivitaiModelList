### 三、详情
<p>This version includes 2 LoRA, one for the character+outfits and another for Ayakura Juu's style: <a target="_blank" rel="ugc" href="https://twitter.com/haino">https://twitter.com/haino</a></p><p>CivitAI does not allow multiple LoRA in one version so I've put the style LoRA in the "training data" zip file. I've included example images both with and without the style; refer to the prompts to see how to use the LoRa together. All images are generated using Grapefruitv4.1.</p><p>The character LoRA was trained on NovelAI with 2000 steps and the style LoRA with 3000 steps. Best strength for both LoRA is about 0.5-0.7. Style does not require trigger words. I find the generations are most accurate when including "sash" and "pouch" tags in the prompt. Also try increasing the strength of the trigger words e.g. (holo:1.5).</p><p>Feel free to share your images.</p>
