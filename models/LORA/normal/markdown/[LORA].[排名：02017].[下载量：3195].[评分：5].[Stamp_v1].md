### 三、详情
<p>Some patterns like stamps can be generated. Common words can be: moon, mountain, bird, rose...</p><p></p><p>【tips】</p><p>First you need to install the lora extension, and then put the downloaded stampV1_v10.safetensors file into the .\extensions\sd-webui-additional-networks\models\lora folder.</p><p>When using it, click the lora plug-in tab (the tab may be located under the seed), check the enable option, and select the stampV1_v10 model, and adjust the intensity from 0.6 to 1 according to your needs.</p><p>For the base model in the upper left corner of Stable Diffusion, select the original default one.</p><p>Regarding the strength: If you want the picture to be more like a graphic design, you can make it stronger, such as setting it to 1. But if you want richer elements such as animated images in the picture, you can set it lower, such as 0.7</p>
