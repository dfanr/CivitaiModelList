### 三、详情
<p><strong>LoRA for generating images with the "Symbiote / She Venom" Character</strong></p><p></p><p>LoRA request queue: <a target="_blank" rel="ugc" href="https://docs.google.com/spreadsheets/d/1DXjs-96citoEa4NPWWxm4o92kQyrJpgdXQGWQg65oiM/edit?usp=sharing">https://docs.google.com/spreadsheets/d/1DXjs-96citoEa4NPWWxm4o92kQyrJpgdXQGWQg65oiM/edit?usp=sharing</a></p><p></p><p>This model was trained using the NAI model. A lot of them were NSFW, so there is a good chance you'll get those as a result. I made 2 models prior to this and merged them together to create this one since the first one was not good enough for me and the second one wasn't that accurate with the white pattern.</p><p></p><p>Works best by itself but with some inpainting you can get it to work with other character LoRA's.</p><p></p><p>I got the best results with a weight between <strong>0.6 and 0.8.</strong></p><p></p><p>Trigger word is <em>symbiote, </em>if you only use that you will get the symbiote face, but since it was trained on other faces aswell you can specify what else you want and it will show a normal face.</p><p></p><p>Example images were generated using Anything V4.5 on Euler A with 30 steps.</p>
