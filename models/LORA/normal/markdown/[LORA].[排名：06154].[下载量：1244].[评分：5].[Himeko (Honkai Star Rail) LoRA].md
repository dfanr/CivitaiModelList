### 三、详情
<h1>Welcome to the Astral Express!~</h1><p></p><h2>General Info</h2><p>I know there are already existing Himeko LoRA's on CIvitai but they did not exactly achieve what i was looking for so i trained my own. The big advantage of this LoRA is that you only need a single trigger word to generate quality images of her. It also allows for different styles and poses of her. This LoRA is also trained on the Aurora model from Luna (<a target="_blank" rel="ugc" href="https://civitai.com/models/40199/aurora">https://civitai.com/models/40199/aurora</a>), the example images have been generated using that model, too.</p><h2>Usage</h2><p>The model has been trained using Kohya so you probably want to use it in the same way as you use other Kohya models. If you are Using A1111's Webui you want to move the file into the models&gt;Lora directory. Make sure to use the corresponding trigger words to get the expected results. Also i recommend a LoRA weight of around 0.8.</p>
