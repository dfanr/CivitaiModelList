### 三、详情
<p>LoRA trained on the animefull-final model using the official artwork of the mainline Pokémon games. I'd personally recommend using this the most with AnythingV3.</p><p></p><p>For full body shots I'd suggesting using a resolution of 448x896, as this was the largest bucket in the dataset. Most resolutions still work quite well though and which you want to use can depend a lot on the pose/character you want to depict.</p><p></p><p>All artwork has been tagged with their respective generation (eg. generation 1, generation 2, etc.). Allowing you to get different styles, occasionally there's a bit of bleeding (mostly at gens 3 and 4) but its fairly consistent otherwise.</p><p></p><p>As for upscaling, I recommend using something like the anime ESRGAN models, rather than the default latent upscaling. As the Pokémon artstyle doesn't have a lot of details any new stuff added usually looks bad.</p>
