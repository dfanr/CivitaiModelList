### 三、详情
<p>I trained this DreamBooth model using EveryDream. The model was trained using the art from artist Tokkyu at 57 images, for 3 epochs in a total of 1440 steps. I trained it on a A100 (80gb Vram) with 768x768 native size/variable buckets, batch size 6, at around 68gb Vram per epoch using very high resolution training images non cropped. EveryDream handled the aspect ratios correctly. The images were captioned manually each with a little help from pharmapsychotic's excellent new 2.1 CLIP Interrogator.</p><p>There is a visual guide explaining several aspects of the model and how you can get the highest quality in your txt2img generations.</p><p>The model requires an Aesthetic Gradient to get the highest level of quality.
