### 三、详情
<p>This model was trained on the dataset with kawakaze images and then merged with other my models.</p><p>The model itself can produce other characters besides kawakaze, but sometimes there is a strong bias in the character traits, so I decided to name the model that way.</p><p>All images in the examples used lora with multiplier 0.4, this lora is in my profile.</p><p>And also, all the examples are made with multidiffusion, so the original seeds are lost.</p><p></p>
