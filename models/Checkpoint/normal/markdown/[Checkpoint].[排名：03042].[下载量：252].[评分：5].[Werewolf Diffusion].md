### 三、详情
<p>Creates werewolves, and can be used to create other were-things if you do not use the keyword. I didn't see any model that could make werewolves quite well enough and I wanted to be able to generate some digital-art style werewolves from which I could then paint over, and this model serves that purpose well for me. Far from perfect and is definitely a bit overcooked, but can generate some cool werewolves. Trained on SD 1.5 with a custom dataset.</p><p>Use 'wewulzkz' as the keyword. </p><p>Here is an example of an finished image I created using this model:</p><p><a target="_blank" rel="ugc" href="https://twitter.com/ArtifartX/status/1619179667503857665?s=20">https://twitter.com/ArtifartX/status/1619179667503857665?s=20</a></p><p>And here are some other raw outputs of NOT using the keyword, and then typing in other animals, as it does kind of a cool job turning them into were-animals (so for these I just used different animal names in the prompt, and did NOT use 'wewulzkz' and it seems to apply the werewolf idea to them decently).</p><p>Mouse/Cat:</p><p><a target="_blank" rel="ugc" href="https://i.imgur.com/CGvMl9e.png">https://i.imgur.com/CGvMl9e.png</a></p><p>Alligator:</p><p><a target="_blank" rel="ugc" href="https://i.imgur.com/zJWU004.png">https://i.imgur.com/zJWU004.png</a></p>
