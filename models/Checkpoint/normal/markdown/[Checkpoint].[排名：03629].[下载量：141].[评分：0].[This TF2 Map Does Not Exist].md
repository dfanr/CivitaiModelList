### 三、详情
<p>This is my first model and i have no idea what i'm doing, but it makes some pretty cool TF2 map renders for the 0 effort i put into it.</p><p></p><p>Generates mostly renders of mid-points of a TF2 5cp map. Might be useful for inspiration if you're a map maker. Will look like in-engine renders unless you add something like "in the artstyle of norman rockwell" to your positive prompt. Trained on 512x384, based on dreamshaper.</p><p></p><p>There is a LOT of room for easy improvements here and if people are interested i'll probably make a much better v2 of this in the future. Training data is included (random album of 72 map screenshots i found on imgur)</p>
