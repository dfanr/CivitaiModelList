### 三、详情
<p><strong>AI Beast Mix</strong> is probably one of my greatest model merge to achieve semi-realism result for furry character. The model mainly focus on male/ gay furry character but it can also generate female character too. This model seem to be able to support a lot of lora during my testing. However, I don't recommend using this model to train character lora but if train for concept, it might work but I'm not too sure about it.</p><p></p><p>Unlike my old model <strong>Beast Mix</strong>, this model have lower chance generate weird/ curse result. Also, <strong>Hires.Fix</strong> is recommended to get better details and result.</p><p></p><p>If I'm not mistaken, this model does not use any<strong> VAE </strong>so don't worry about it. If it does, it's already baked with one of the model during merge process.</p>
